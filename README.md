# Gleec × Gnosis Pay — Financial Model Note

An executive briefing that explains the Gleec / Gnosis Pay card financial model (**v48**) in plain language, with interactive, hoverable charts of the 24-month P&L, break-even, revenue/cost mix, a before-and-after model comparison, and the cash-need profile.

**Live:**
- Vercel — https://gleec-gnosis-financial-model-note.vercel.app
- GitHub Pages — https://elijah-klinkin.github.io/gleec-gnosis-financial-model-note/

## What's here

A single, self-contained `index.html` — all styles, scripts, charts, and interactivity are inlined. No build step, no dependencies, no external requests. It renders in both light and dark themes and is fully responsive.

| File | Purpose |
|------|---------|
| `index.html` | The briefing (self-contained) |
| `vercel.json` | Static hosting config (clean URLs) |

## Highlights of the v48 model

- Break-even in **month 5**; cumulative losses recovered by **month 9**
- Month-24 run-rate **+$230k/mo**; 24-month cumulative **+$2.2M**
- Driven by repricing the customer top-up fee from 0.25% → **1.75%** (net top-up margin 1.50%)
- Peak cash need **~$2.5M**, now essentially all collateral

## Notes

Figures are model outputs under stated assumptions — not forecasts or investment advice. Interchange is deliberately conservative under UK PSR fee caps; the programme is assessed through the Canadian MSB entity.
